Explanation

This example program is from the curriculum at Angular Boot Camp, An in-depth Angular class offered by Oasis Digital.

The project is compiled using:

  • AOT
  • the new experimental es2015 ESM packaging of the Angular libraries
  • Rollup
  • Buble
  • Uglify

See the project README for more explanation.

In this example, we use Observable operators to efficiently and correctly call a remote API. The API chosen is a Reddit image search. To use it, edit the search string field contents with something like "meow".

In the browser JavaScript console, you can see messages as the search criteria change, you can see the change denounced, and see the results return.
